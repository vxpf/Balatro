Balatro C# Project

Over dit project
Dit project is een eigen versie van het kaartspel Balatro gemaakt in C#. Het doel van dit project is leren werken met object oriented programming, game logica en software ontwerp. Het project wordt gebruikt als bewijs voor leerdoelen richting stage.

Technologie
Programmeertaal: C#
Ontwikkelomgeving: Visual Studio
Type project: Game development, schoolproject

Project doelen
• Werken met klassen en objecten in C#
• Game logica bouwen
• Diagrammen gebruiken voor ontwerp
• Software project documenteren
• Voorbereiding op stage

Game concept
Balatro is een kaartspel gebaseerd op poker combinaties. De speler speelt rondes en probeert een hoge score te halen met kaarten en combinaties. Veel versies van Balatro draaien om strategie met kaarten en combinaties. Balatro
